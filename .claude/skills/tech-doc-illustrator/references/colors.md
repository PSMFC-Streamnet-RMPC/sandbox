# Color Palette Reference

Single source of truth for all illustration colors.

## Accent Colors

Use ONE accent color per illustration. The accent color provides emphasis and visual hierarchy.

| Color   | Hex       | RGB             | Best For                                      |
|---------|-----------|-----------------|-----------------------------------------------|
| Coral   | `#e85d3b` | 232, 93, 59     | Energy, action, important highlights, errors  |
| Teal    | `#0d9488` | 13, 148, 136    | Technology, connectivity, data flow, success  |
| Indigo  | `#6366f1` | 99, 102, 241    | Intelligence, depth, advanced concepts, info  |
| Amber   | `#eab308` | 234, 179, 8     | Warnings, attention, configuration, caution   |

## Base Colors

These colors form the foundation of every illustration:

| Element         | Color          | Hex       | Notes                                |
|-----------------|----------------|-----------|--------------------------------------|
| Background      | Warm off-white | `#faf9f6` | Subtle warmth, not pure white        |
| Primary lines   | Confident black| `#1a1a1a` | Strong but not harsh                 |
| Secondary lines | Soft gray      | `#6b7280` | Supporting elements, depth           |
| Shadow/depth    | Cool gray      | `#9ca3af` | Use sparingly (5-10% of composition) |

## Usage Rules

1. **ONE accent color per illustration** - Never mix accent colors
2. **Accent color is sparse** - 10-20% of the illustration maximum
3. **Lines are always black or gray** - Accent color for fills only
4. **Background is always warm off-white** - Provides cohesion across all images
5. **No gradients** - Flat colors only for clean, printable results

## Choosing the Right Accent Color

### Coral (#e85d3b)
- Error states or failure scenarios
- Call-to-action elements
- Critical paths in flowcharts
- Important warnings that need immediate attention
- "Hot" or active processes

### Teal (#0d9488)
- Success states or completion
- Data flow and connectivity
- Network diagrams
- API connections
- Database relationships
- "Cool" or stable systems

### Indigo (#6366f1)
- AI/ML concepts
- Advanced or complex features
- Security and authentication
- Configuration management
- Abstract technical concepts

### Amber (#eab308)
- Caution or warning states (non-critical)
- Configuration settings
- Pending or in-progress states
- User attention needed
- Optional but recommended actions
